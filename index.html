Pengertian Komponen Cara Kerja Jaringan Sistem Operasi Tips Belajar

1. Pengertian Komputer

Komputer adalah perangkat elektronik yang mampu menerima data, memprosesnya, dan menghasilkan informasi yang berguna. Komputer dapat digunakan untuk mengetik, bermain game, browsing, hingga membuat program.

2. Komponen Komputer

Komputer tersusun dari Hardware dan Software:

  • Hardware - Bagian fisik: CPU, RAM, GPU, keyboard, monitor
  • Software - Program: Windows, Linux, Photoshop, Game

Selain itu ada Brainware yaitu manusia yang mengoperasikan komputer.

3. Cara Kerja Komputer

  1. Input - Masukan data via keyboard, mouse, scanner
  2. Process - CPU memproses data
  3. Output - Hasil ditampilkan di monitor atau printer
  4. Storage - Data disimpan di HDD/SSD

4. Jaringan Komputer

Jaringan komputer menghubungkan komputer agar bisa berbagi data & sumber daya.

  • LAN (Local Area Network) - Skala lokal, lab sekolah
  • MAN (Metropolitan Area Network) - Skala kota
  • WAN (Wide Area Network) - Skala dunia, Internet
  • WLAN - LAN tanpa kabel (WiFi)

Topologi Jaringan:

  1. Star - Semua PC ke switch/hub (paling sering)
  2. Bus - Satu kabel utama
  3. Ring - Bentuk lingkaran
  4. Mesh - Semua saling terhubung

5. Sistem Operasi

Sistem operasi adalah software utama yang mengatur hardware dan software lain.

  • Windows - Mudah dipakai, populer
  • Linux - Gratis, open source, ringan
  • MacOS - Eksklusif untuk Apple

6. Tips Belajar Komputer untuk Pemula

  • Belajar sedikit tapi rutin setiap hari
  • Sering praktek langsung, bukan teori doang
  • Coba ngoding HTML/JS untuk latihan logika
  • Jangan takut error, karena dari error kita belajar
